### パラメータ
- [x] 投手の球速パラメータ（平均球速、±5kmの変動）
- [x] 球速による結果確率の補正
- [x] 投手の制球力パラメータ（1〜10段階）
  - 制球力が高いほどボール球が減少（25%〜45%の範囲）
  - 制球力が高いほど甘い球が減り、被打率が低下
- [x] 野手のミート力パラメータ（1〜10段階）
  - ミート力が高いほど空振りが減少 → インプレー（打球が飛ぶ）確率が上昇
  - インプレー後のヒット/アウト判定には影響しない（バットに当てる力）
- [x] 野手の長打力パラメータ（1〜10段階）
  - 長打力が高いほどホームラン確率が大幅に上昇
  - 長打力が高いほど二塁打・三塁打確率も上昇
  - 長打力が高いほど単打もわずかに増加（打球速度）
- [x] 打球方向の概念追加
  - FieldPosition列挙型（投手/捕手/一塁/二塁/三塁/遊撃/左翼/中堅/右翼）
  - 打球の種類（ゴロ/フライ/ライナー）によって飛ぶ方向の確率が変化
  - ゴロは内野、フライは外野メイン、ライナーは全方向に分散
- [x] 守備力パラメータ（ポジションごと、0〜10）
  - DefensePosition列挙型（捕手/一塁/二塁/三塁/遊撃/外野）
  - 外野は左翼・中堅・右翼をまとめて1つ
  - 0はそのポジションを守れない、1〜10は守備力
  - 守備力が高いほどヒットになりにくい（アウト率が上がる）
  - チームに守備配置（誰がどこを守るか）を設定可能
