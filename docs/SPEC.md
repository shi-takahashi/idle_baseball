# 放置系プロ野球GMになろう！ - 仕様書

## コンセプト

架空のプロ野球チームのオーナー兼GMとして、チームを運営する放置系シミュレーションゲーム。
ベストプレイプロ野球のエッセンスを残しつつ、アニメーションを省略し、放置系の要素（時間による変化）を加えた設計。

---

## 1. ゲームの基本仕様

### 1.1 試合の進行

| 項目 | 仕様 |
|------|------|
| 試合タイミング | 毎日決まった時間に自動で1試合実行 |
| 結果確認 | 設定時刻以降にユーザーが結果を確認 |
| 表示形式 | スコアボード形式（新聞風） |

### 1.2 リーグ構成

| 項目 | 仕様 |
|------|------|
| チーム数 | 1リーグ6チーム |
| シーズン試合数 | 基本30試合（設定で143試合も可能） |
| シーズン期間 | リアルタイムで約1ヶ月（30日） |

### 1.3 試合結果の表示

- **スコアボード**: 各イニングの得点を新聞形式で表示
- **個人成績**: 打撃成績、投手成績を表示
- **詳細表示**: 各イニングの表/裏を選択すると、打席ごとの対戦詳細を確認可能
  - どの投手 vs どの打者
  - 結果（ヒット、三振、ホームラン等）

## 2. 選手パラメータ

### 2.1 評価方式

- 10段階評価（1〜10）
- 年齢と成長タイプにより毎シーズン能力が変動

### 2.2 野手パラメータ（案）

| パラメータ | 説明 |
|-----------|------|
| ミート | バットにボールを当てる能力 |
| 長打力 | 長打を打つ能力 |
| 走力 | 走る速さ |
| 守備力 | 守備の上手さ |
| 肩力 | 送球の強さ・正確さ |

### 2.3 投手パラメータ（案）

| パラメータ | 説明 |
|-----------|------|
| 球速 | ストレートの速さ |
| 制球 | コントロールの良さ |
| 変化球 | 変化球の切れ |
| スタミナ | 長いイニングを投げる能力 |
| 精神力 | ピンチに強いか |

### 2.4 共通パラメータ（案）

| パラメータ | 説明 |
|-----------|------|
| 年齢 | 現在の年齢 |
| 成長タイプ | 早熟/普通/晩成 |
| ポジション | 野手: C/1B/2B/3B/SS/LF/CF/RF、投手: SP/RP/CP |

## 5. 収益化モデル

### 5.1 広告（メイン）

- バナー広告
- リワード動画広告（任意視聴）

### 5.2 課金オプション（月額サブスクリプション）

| プラン | 内容 | 想定価格 |
|--------|------|---------|
| ライト | 広告削除のみ | 月額100〜200円 |
| スタンダード | 広告削除 + 一部機能 | 月額200〜300円 |
| プレミアム | 全機能（広告削除 + エディット + 時間スキップ） | 月額500円程度 |

**含まれる機能**:
- 広告削除: 広告を非表示
- チームエディット: 選手名・能力を自由に編集可能
- 時間スキップ: 複数試合を連続で進行可能

**チームエディット機能の狙い**:
- ユーザーが現実のプロ野球選手データを自分で編集して遊べる
- 公式データではないため版権問題を回避
- 「自分だけのプロ野球シミュレーション」として価値提供

### 5.3 課金基盤

- **RevenueCat** を使用
- 自前サーバー不要
- Google Play / App Storeの課金をRevenueCatが代行
- 月間収益$2,500以下は無料で利用可能

## 8. 成績記録・タイトル

### 8.1 チーム成績

| 項目 | 説明 |
|------|------|
| 勝敗 | 勝ち、負け、引き分け |
| 勝率 | 勝利数 / (勝利数 + 敗北数) |
| ゲーム差 | 首位とのゲーム差 |
| 得点 | チーム総得点 |
| 失点 | チーム総失点 |
| 得失点差 | 得点 - 失点 |

### 8.2 野手個人成績

| 項目 | 説明 |
|------|------|
| 打率 | 安打数 / 打数 |
| 試合数 | 出場試合数 |
| 打席数 | 打席に立った回数 |
| 打数 | 打席数 - 四死球 - 犠打等 |
| 安打 | ヒット数 |
| 二塁打 | 二塁打数 |
| 三塁打 | 三塁打数 |
| 本塁打 | ホームラン数 |
| 打点 | 打点数 |
| 得点 | ホームを踏んだ回数 |
| 盗塁 | 盗塁成功数 |
| 四球 | フォアボール数 |
| 三振 | 三振数 |
| 出塁率 | (安打 + 四死球) / (打数 + 四死球 + 犠飛) |
| 長打率 | 塁打数 / 打数 |
| OPS | 出塁率 + 長打率 |

### 8.3 投手個人成績

| 項目 | 説明 |
|------|------|
| 防御率 | (自責点 × 9) / 投球回 |
| 登板数 | 登板した試合数 |
| 先発 | 先発登板数 |
| 勝利 | 勝ち数 |
| 敗北 | 負け数 |
| セーブ | セーブ数 |
| ホールド | ホールド数 |
| 投球回 | 投げたイニング数 |
| 被安打 | 打たれたヒット数 |
| 被本塁打 | 打たれたホームラン数 |
| 奪三振 | 三振を取った数 |
| 与四球 | 与えたフォアボール数 |
| 失点 | 失った点数 |
| 自責点 | 自分の責任による失点 |
| WHIP | (与四球 + 被安打) / 投球回 |

### 8.4 タイトル

シーズン終了時に各タイトルを決定し表彰：

**野手タイトル**

| タイトル | 条件 |
|---------|------|
| 首位打者 | 打率1位（規定打席以上） |
| 本塁打王 | 本塁打数1位 |
| 打点王 | 打点数1位 |
| 盗塁王 | 盗塁数1位 |
| 最高出塁率 | 出塁率1位（規定打席以上） |

**投手タイトル**

| タイトル | 条件 |
|---------|------|
| 最優秀防御率 | 防御率1位（規定投球回以上） |
| 最多勝 | 勝利数1位 |
| 最多奪三振 | 奪三振数1位 |
| 最多セーブ | セーブ数1位 |
| 最優秀中継ぎ | ホールドポイント1位 |

**MVP・その他**

| タイトル | 条件 |
|---------|------|
| MVP | シーズン最優秀選手（アルゴリズムで算出） |
| 新人王 | 1年目選手の最優秀者 |
| ベストナイン | 各ポジション最優秀選手 |

### 8.5 規定打席・規定投球回

| 項目 | 計算式（30試合シーズンの場合） |
|------|-------------------------------|
| 規定打席 | 試合数 × 3.1 = 93打席 |
| 規定投球回 | 試合数 × 1.0 = 30イニング |

※ 143試合設定時は自動で調整

### 8.6 通算成績

- 選手ごとに通算成績を保存
- 複数シーズンをまたいで積み上げ
- 引退時に「通算本塁打○本」等を表示可能
